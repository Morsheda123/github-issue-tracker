1. What is the difference between var, let, and const?

Ans: var, let , and const are mainly used to declare variables in JavaScript.The difference between var, let, and const is that var is the old way and it is function-scoped, so it can be accessed outside blocks like if or loops. let is block-scoped and its value can be changed later. const is also block-scoped but its value cannot be reassigned after it is declared.

2. What is the spread operator (...)?

Ans: spread operator (...) is  used to expand elements of an array or object. It helps to copy or combine arrays and objects easily.

3.  What is the difference between map(), filter(), and forEach()?

Ans: The difference between map(), filter(), and forEach() are given below
 ->map() creates a new array by applying a function to each element.
 ->filter() creates a new array with elements that match a condition.
 ->forEach() only runs a function for each element but does not return a new array.

 Ans:Template literals are a way to write strings in JavaScript using backticks instead of quotes. They allow us to insert variables inside a string using ${}.
